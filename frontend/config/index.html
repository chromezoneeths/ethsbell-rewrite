Configuration Helper (Simple.)

You can use the configuration helper to configure the software graphically.





---
When you visit ETHSBell at its default page (the root) you will be redirected to the page you select here. If you want to override this behavior, add "?no_default=true" to your link.
This behavior will not occur when you arrive through a link on ETHSBell.
---

Note: Attempting to build your own dark mode with these settings isn't likely to work, but the Dark Reader browser extension does an excellent job when set to static mode.






Write a schedule manually (Advanced!)

You can make your own schedule file with a standard text editor. A schedule file should look like this, omitting content after "//". 

{
	"schedule": {
		"Block1": {
			"name": "Your First Period Class",
			"url": "https://meet.google.com/whatever"
		},
		"Block3": {
			"name": "Second Period Class",
			"url": "https://classroom.google.com/whatever"
		}
		"Block3": {
			"name": "Class With No Teleconferencing or Classroom Link",
			"url": undefined
		}
		...
	},
	"foreground_color": "#1a2741",
	"background_color": "#c34614",
	"foreground_text_color": "#ffffff",
	"background_text_color": "#ffffff",
	"default_page": "/timeline" // This can be set to any URL, but ideally it will point to a path under our domain.
}
