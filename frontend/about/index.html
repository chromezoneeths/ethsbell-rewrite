About ETHSBell NG

ETHSBell NG is a next-generation rewrite of ETHSBell from scratch in Rust, a modern compiled language. It has new features, a better user interface, a faster and more secure backend, a smaller network footprint, and a permissive license.

ETHSBell NG is written by the CodETHS club. Its source code is available here, and dynamic documentation for writing your own client software is available here. It is published under the MIT License.

To report an issue with ETHSBell NG, contact Willow Carlson-Huber for server-side issues or Albert Portnoy for issues with the user interface.
